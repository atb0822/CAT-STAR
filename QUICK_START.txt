WHAT IS CAT-STAR 2000?
----------------------
CAT-STAR 2000 is a professional cable access television information system 
that displays community events, weather forecasts, and announcements in a 
classic broadcast channel format.

Perfect for:
- Cable access TV channels
- Community information displays
- Municipal information systems
- Public access channels

================================================================================
INSTALLATION
================================================================================

STEP 1: PREREQUISITES
---------------------
You need Node.js installed on your computer.

Download from: https://nodejs.org/
Recommended version: 18.x or higher

STEP 2: EXTRACT FILES
----------------------
Extract the CAT-STAR 2000 ZIP file to a folder of your choice.
Example: C:\CAT-STAR or /home/user/CAT-STAR

STEP 3: START THE SERVER
-------------------------
Windows:
   Double-click START_SERVER.bat

Linux/Mac:
   Open terminal in the folder
   Run: ./START_SERVER_LINUX.sh

The script will:
1. Clear npm cache
2. Install all required dependencies automatically
3. Start the CAT-STAR 2000 server

STEP 4: ACCESS THE SYSTEM
--------------------------
Open your web browser and go to:

   http://localhost:3000/menu

This is your main menu where you can access all features.

================================================================================
TROUBLESHOOTING
================================================================================

SERVER WON'T START:
- Make sure Node.js is installed
- Run the startup script (it installs dependencies)
- Check port 3000 isn't already in use

CAN'T ACCESS WEB INTERFACE:
- Make sure server is running
- Try http://127.0.0.1:3000/menu instead
- Check firewall settings

BACKUP BUTTON NOT WORKING:
- The startup script installs required packages
- If still not working, manually run: npm install

WEATHER NOT LOADING:
- Check internet connection (for NOAA data)
- Verify ZIP code is valid
- Check coordinates are correct

ROTATION NOT WORKING:
- Save settings in Admin Panel
- Reload the display page
- Clear browser cache (Ctrl+F5)
- Check browser console (F12) for errors
